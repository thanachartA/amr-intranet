# AMR Asia Intranet Portal

Employee self-service portal for AMR Asia PCL.

## Tech Stack
- Next.js 14 + TypeScript + Tailwind CSS
- - Supabase (Auth + PostgreSQL)
  - - Vercel (Deployment)
   
    - ## MVP Modules
    - - **HRM**: Benefits, Documents, Certificates, Regulations, Complaints
      - - **General Admin**: Supplies, Equipment, Vehicles, Rooms, Maintenance
